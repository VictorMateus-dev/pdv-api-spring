# 🛒 PDV API — Sistema de Ponto de Venda

API REST para gerenciamento de um sistema de mercado (PDV), desenvolvida com Spring Boot. O projeto está em desenvolvimento ativo.

---

## 🚀 Tecnologias

- **Java 17**
- **Spring Boot 4.x**
- **Spring Data JPA**
- **Spring Security** *(em desenvolvimento)*
- **Spring Validation**
- **MySQL**
- **Lombok**
- **Maven**
